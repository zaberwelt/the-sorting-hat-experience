# the-sorting-hat-experience
Interactive Sorting Hat simulator with Hogwarts theme.  
Features:
- Colorful houses: Gryffindor, Hufflepuff, Ravenclaw, Slytherin
- Animated Sorting Hat and sparkling effects
- House icons appear after sorting
- Aesthetic Hogwarts background

## How to Use
1. Clone this repository
2. Open `index.html` in your browser
3. Click "Sort Me!" to be sorted into a house
